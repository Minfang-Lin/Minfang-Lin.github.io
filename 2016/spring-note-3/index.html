实现Controller

定义Controller

@RequestMapping

  • name: 名称
  • value & path： 路径，如”/hello“
  • method: 请求方法，如”GET”
  • params: 请求参数
  • headers： 请求头，如Accept头
  • consumes： 请求的媒体类型，”Content-Type”
  • product： 响应的媒体类型，”Accept”
